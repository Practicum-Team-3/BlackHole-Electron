Types of Panels

Scenarios Panel Boxes Panel Exploits Panel Programs Panel Network Graph Settings Panel

Scenarios Panel

This panel contains all of the created scenarios.

Boxes Panel

This panel contains all of the boxes created.

Exploits Panel

This panel contains all of the exploits uploaded to the system.

Programs Panel

This panel contains all of the programs uploaded to the system.

Network Graph

This panel is a graph that contains the connected nodes that represent boxes added to a scenario.

Settings Panel

This panel contains the editable Settings that can be done to a Box.
